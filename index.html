Electronic ID verification and extraction

Hutts Verification is an open source, Python-based Web API for electronically verifying and extracting information from images of South African IDs.

Live Demo

Services

Hutts Verification provides two main services, namely extraction and verification of South African ID information.
A request for said services, and their responses, are in JSON.

Extraction

Verification

Extraction is a service provided for extracting information from an individual's ID and can be used for purposes such as electronically completing forms.

Three types of extraction requests can be performed:

  • Extraction of an individual's personal details only
  • Extraction of an individual's photo only
  • Extraction of an individual's personal details and photo

Verification is a service provided for electronically verifying information from an individual's ID against existing information, so as to aid in the confirmation of an individual's identity.

Three types of verification requests can be performed:

  • Verification of an individual's personal details only
  • Verification of an individual's photo only
  • Verification of an individual's personal details and photo
